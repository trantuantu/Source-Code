USAGE: Mean stack Example
SPECS:
Backend: NodeJS (Express framework)
Front-end: Javascript (AngularJS framework)
Database: Mongod
Technologies: JSON Web token Authentication (Web service Auth Protection), Hash Password, Responsive Web, Boostrap, Ajax, JQuery
START PROGRAM:
Install NodeJS, Mongod, npm
Step 1: Create directory C:\data\db
Step 2: Open cmd and use this command to open mongod: mongod --storageEngine=mmapv1
Step 3.1: In this folder use: npm install
Step 3.2: In this folder use: npm start
Step 4: Open browser in: http://localhost:3000/
