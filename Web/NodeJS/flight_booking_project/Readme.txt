DEMO:
Demo video: https://www.youtube.com/watch?v=vHwGaJNWInU
Host demo: http://cnm-flight-booking.herokuapp.com/
USAGE:
This application simulate flight booking page
You can use mobile app in Mobile/QRCodeFlightBooking to Scan QRCode generated by this Application
SPECS:
Host: Heroku
Backend: NodeJS (Express framework)
Front-end: Javascript (AngularJS framework)
Database: MySQL
Technologies: Docker, Socket (Auction realtime), JSON Web token Authentication (Web service Auth Protection), Hash Password, Responsive web, Boostrap, JQuery (Slide show), AJax, API Restful Web service, Local Storage
START PROGRAM:
import database in database folder
Run these comment:
	npm install
	node app.js
Go to localhost:3000 to show the page
